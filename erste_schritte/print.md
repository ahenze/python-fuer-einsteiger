
# Ausgabe auf den Bildschirm

Im diesem Teil werden wir den `print`-Befehl zur einfachen Textausgabe weiter erkunden.


### Aufgabe 1

Welche `print`-Anweisungen sind korrekt?

* `print("9" + "9")`
* `print "nine"`
* `print(str(9) + "nine")`
* `print(9 + 9)`
* `print(nine)`

### Aufgabe 3
